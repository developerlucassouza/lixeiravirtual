Tipos Primitivos do MySQL

O MySQL possui quatro grupos principais de Tipos Primitivos: Literal, Numérico, Data/Tempo e Espacial.

Literal

  • Caractere
    • Char
    • VarChar
  • Texto
    • TinyText
    • Text
    • MediumText
    • LongText
  • Binário
    • TinyBlob
    • Blob
    • MediumBlob
    • LongBlob
  • Coleção
    • Enum
    • Set

Os mais utilizados são: Char, Varchar e Text.

Tipos Numéricos

  • Inteiro
    • TinyInt
    • SmallInt
    • Int
    • MediumInt
    • BigInt
  • Real
    • Decimal
    • Float
    • Double
    • Real
  • Lógico
    • Bit
    • Boolean

Os mais utilizados são: TinyInt, Int, Float, Decimal e Bit.

Data/Tempo

  • Date
  • DateTime
  • TimeStamp
  • Time
  • Year

Espacial

  • Geometry
  • Point
  • Polygon
  • MultiPolygon
