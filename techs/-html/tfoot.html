Tag <tfoot> HTML

A tag tfoot é utilizada para criar o rodapé de uma tabela (<table>), isto é, a parte final da tabela.

<table>
    <thead>
        [...]
    </thead>

    <tbody>
        [...]
    </tbody>

    <tfoot>
        <tr>
            <th>Total</th>
            <td>100.000</td>   
        </tr>
    </tfoot>
</table>

Tal tag deve ser usada em conjunto das tags <thead> e <tbody>.
