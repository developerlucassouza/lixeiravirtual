Tag <tr> HTML

A tag tr é utilizada em tabelas (<table>) para indicar uma linha da tabela.

Dentro das tags tr, são utilizadas as tags <td> para inserir os dados.

<table>
    <!-- Primeira Linha -->   
    <tr>
        <td>B1</td>
        <td>A1</td>
        <td>C1</td>
    </tr>

    <!-- Segunda Linha -->
    <tr>
        <td>A2</td>
        <td>B2</td>
        <td>C2</td>
    </tr>
</table>
