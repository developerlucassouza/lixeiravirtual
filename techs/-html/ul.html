Tag <ul> HTML

A tag ul cria uma lista não ordenada, isto é, uma lista que não possui uma ordem correta.

Para adicionar cada item da lista é preciso utilizar a tag <li>.

<ul>
    <li>Item 1</li>   
    <li>Item 2</li>
    <li>Item 3</li>
    <li>Item 4</li>
</ul>
