Instalação está no Bloco de Notas do Bootstrap hehe

=====================================================================================

Dentro do seu arquivo .js, com o jQuery já importado, temos q montar a seguinte estrutura para mecher com o jQuery:

jQuery(document).ready(function($) {
	// Aqui é escrito o código
})

=====================================================================================

Mudar CSS de acordo o querySelector():
$('img.algo').css('margin','100px auto')

=====================================================================================

$('.btnProcurar').on('click', clicouProcurar)

function clicouProcurar(){
	let txtdigitado = $('#txtDigitado').val()
        window.alert(txtdigitado)
