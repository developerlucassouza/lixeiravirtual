Output em Python

Output é quando você mostra algo ao usuário, é a saída de dados de um programa.

Em Python, a função para saída de dados é o print().

Dentro dos parenteses, você irá escrever aquilo que quer escrever ao usuário.

Vamos começar com um clássico Olá, Mundo!

print("Olá, Mundo!")
Olá, Mundo!

Peceba que o texto está escrito entre aspas.

As aspas indicam um texto, algo do tipo primitivo string, mas isso será abordado mais a frente.

Podemos usar o print() para executar contas matemáticas.

Por não se tratar de texto, e sim números e cálculos, não será preciso utilizar aspas.

print(4+4)
8

O uso das aspas faz com que seja mostrado o texto de forma literal.

print("4+4")
4+4

Além disso, podemos usar o sinal de soma (+) para concatenar (juntar) textos, também chamados de string.

print("Olá," + "Mundo!")
print("4" + "4")
Olá, Mundo!
44

Há diferença entre 4 e "4" (com e sem aspas)

O 4 é do tipo int, isto é, um número inteiro.

O "4" é do tipo string, isto é, um texto.

Por isso não é possível somar "4", pois não é possível somar textos.

Para comentar em Python usamos a hashtag #.

# Este é um comentário em Python
