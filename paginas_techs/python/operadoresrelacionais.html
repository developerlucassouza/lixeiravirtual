Operadores Relacionais em Python

Os Operadores Relacionais comparam dois dados para ver como eles se relacionam.

Eles irão retornar valores booleanos (True ou False).

Exemplo, podemos ver se duas variáveis numéricas são iguais utilizando o operador == (pois como vimos, o operador = é utilizado na atribuição de variáveis.

x = 5
y = 5
print(x == y)   
True

Ou, caso não seja igual:

x = 3
y = 2
print(x == y)   
False

Há vários operados relacionais, veja:

Símbolo Relação
== Igual a
> Maior que
< Menor que
>= Maior ou igual a
<= Menor ou igual a
!= Diferente de
