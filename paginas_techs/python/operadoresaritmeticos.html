Operadores Aritméticos do Python

Como vimos anteriormente, podemos efetuar cálculos de soma com o sinal +, mas podemos efetuar outros cálculos, como subtração, multiplicação, etc.

Símbolo Cálculo Exemplo Resultado
+ Adição 5 + 2 7
- Subtração 5 - 2 3
* Multiplicação 5 * 2 10
/ Divisão 5 / 2 2.5
** Potência 5 ** 2 25
// Divisão Inteira 5 // 2 2
% Resto da Divisão 5 % 2 1

Assim como na matemática, as linguagens de programação possuem uma ordem de precedência:

Ordem Sinais
1° ()
2° **
3° * / // %
4° + -

Multiplicar Strings

Como vimos anteriormente, o sinal + pode ser utilizado para concatenar str.

print('Olá, ' + 'Mundo!')
Olá, Mundo!

Em Python, podemos utilizar o sinal * para repetir uma string.

print('Olá ' * 5)
Olá Olá Olá Olá Olá

Raiz Quadrada

Por padrão, o Python não possui comando para calcular raiz quadrada.

Mas, podemos utilizar a potência de fração para executar uma raiz quadrada.

print( 25 ** (1/2) )
5
