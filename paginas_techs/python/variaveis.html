Variáveis em Python

Variáveis são lugares na memória onde podemos armazenar valores para utilizá-los depois.

Para definir uma variável, utilizamos o sinal de igual (=), que em programação é chamado de recebe.

nome = "Lucas"

Podemos utilizar o comando print() para mostrar uma variável.

print(nome)
Lucas

Também podemos guardar números e fazer cálculos depois.

x = 3
y = 4
print(x + y)
7

Os estudos de variáveis serão mais aprofundados quando falarmos de Tipos Primitivos.
