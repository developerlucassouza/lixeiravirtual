Input em Python

Na página anterior, vimos que podemos exibir mensagens, números e cálculos.

Mas, o que foi visto é apenas mostrar mensagens e cálculos já prontos, não podemos pedir valores ao usuário e mostrá-los.

Digite seu nome: Lucas
Bem-vindo, Lucas!

Para isso, precisamos criar um input, isto é, a entrada de dados pelo usuário.

Para criar uma entrada de usuário, basta utilizar o comando input(), e, dentro dos parênteses, escrever a mensagem que você deseja mostrar.

input("Digite seu nome: ")
Digite seu nome:|

Mas, só isso não basta.

Será preciso guardar esse valor dentro de variáveis. No caso, na variável nome.

Depois, iremos utilizar o sinal de soma (+) para concatenar strings no comando print()

nome = input("Digite seu nome:")
print("Olá, " + nome)
Digite seu nome: Lucas
Olá, Lucas

Mas, se formos utilizar para receber números e somá-los, não irá funcionar.

n1 = input("Digite um número: ")
n2 = input("Digite outro número: ")
print(n1 + n2)
Digite um número: 3
Digite outro número: 4
34

O que acontece é que, o comando input() retorna um valor string por padrão, isto é, um texto. E com texto não se faz cálculo.

No caso, ao invés de fazer 3 + 4, ele está fazendo "3" + "4".

Para poder corrigir isso, precisaremos aprender umm pouco sobre Tipos Primitivos.
