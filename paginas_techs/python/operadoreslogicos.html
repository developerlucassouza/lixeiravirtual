Operadores Lógicos em Python

Os Operadores Lógicos são utilizados para comparar operações lógicas (na maioria das vezes, resultados de Operações Relacionais).

Operador and

O Operador and irá avaliar se todas as condições são verdadeiras, caso seja retorna True, caso não, retorna False.

True and True => True
True and False => False
False and True => False
False and False => False

Exemplos:

5 > 2 and 8 == 8 => True
6 <= 8 and 4 == 9 => False
7 != 7 and 7 == 7 => False
2 < 1 and 3 > 7 => False

Operador or

O Operador or irá avaliar se uma das expressões é verdadeira.

Apenas se todas as expressões forem falsas ele retorna False.

True or True => True
True or False => True
False or True => True
False or False => False

Exemplos:

5 == 5 or 3 == 3 => True
2 == 2 or 9 == 1 => True
4 != 4 or 8 != 2 => True
6 > 9 or 8 < 3 => False

Operador not

O operador not funciona com apenas uma expressão, tornando expressões True em False, e False em True.

not True => False
not False => True

Exemplos:

not 5 == 5 => False
not 7 == 9 => True
