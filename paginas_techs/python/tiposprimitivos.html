Tipos Primitivos do Python

Como já vimos antes, as aspas indicam um texto, ou seja, um dado do tipo string.

Também vimos que um número com e sem aspas possui total diferença.

print(3 + 4)
print('3' + '4')
7
34

Isso acontece por causa do Tipo Primitivo de cada dado.

O Python possui quatro Tipos Primitivos: str (que é o tipo string), int, float e bool.

str: Esse tipo indica palavras e caracteres.

Exemplo: 'Olá, Mundo!', '7.9', 'Python'

int: Esse tipo indica um número inteiro.

Exemplo: 7, -2, 736, 10, 0

float: Esse tipo indica números quebrados, com casas decimais, com vírgula flutuante.

Exemplo: 3.45, 61.7, 4.0, -17.1223

bool: São valores booleanos, valores binários. Eles representam apenas verdadeiro ou falso (true or false)

Exemplo: True, False

Com isso, podemos resolver o nosso problema anterior com os números:

n1 = input('Digite um número: ')
n2 = input('Digite outro número: ')
print(n1 + n2)
Digite um número: 3
Digite outro número: 4
34

O comando input(), por padrão, retorna um valor str, então precisaremos especificar o Tipo Primitivo (no caso, tipo int). Para isso, temos que colocar o comando input() dentro do comando int(). Exemplo:

n1 = int(input('Digite um número: ))

Fazendo essa alteração, nosso código ficará assim:

n1 = int(input('Digite um número: '))
n2 = int(input('Digite um número: '))
print(n1 + n2)
Digite um número: 3
Digite outro número: 4
7

Para entendermos melhor, podemos utilizar o comando type() para vermos o Tipo Primitivo de um dado.

print(type('Olá, Mundo!'))
<class 'str'>

Exemplos:

print(type('Olá, Mundo!'))
print(type(3))
print(type(8.2))
print(type(True))
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>

Também temos o comando .isnumeric(), que diz se uma variável é um número, mesmo se for uma str.

txt = '2'
print(txt.isnumeric)
True

Há alguns comandos semelhantes para outras situações:

.isalpha() para letras.

.isalnum() para letras e números.

.isupper() para letras maiúsculas.

.islower() para letras minúsculas.

.isspace() para espaço.
