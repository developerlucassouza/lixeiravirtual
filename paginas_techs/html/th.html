Tag <th> HTML

A tag th é utilizada para criar uma célula de cabeçalho em uma tabela (<table>)

<table>
    <tr>
        <th>Coluna 1</th>   
        <th>Coluna 2</th>
        <th>Coluna 3</th>
    </tr>

    <tr>
        <td>A1</td>
        <td>B1</td>
        <td>C1</td>
    </tr>
    
    <tr>
        <td>A2</td>
        <td>B2</td>
        <td>C2</td>
    </tr>
</table>
