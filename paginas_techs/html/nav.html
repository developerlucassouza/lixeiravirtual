Tag <nav> HTML

A tag nav é uma tag semântica para indicar um local de navegação, ou seja, uma seção com tags <a> que leva para outras páginas do site.

A tag nav tem apenas função semântica, então não possue estilo padrão.

<nav>
    <a href="[...]">Link 1</a>
    <a href="[...]">Link 2</a>   
    <a href="[...]">Link 3</a>
</nav>
