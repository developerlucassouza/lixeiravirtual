Tag <header> HTML

A tag header é uma tag semântica para indicar o cabeçalho de um site. header não possui nenhum tipo de estilo padrão, mas serve para indicar ao navegador onde fica o cabeçalho de um site e assim estruturar corretamente o seu documento HTML.

<header>
    <h1>Nome do Site</h1>
    <nav>Navegação</nav>
</header>
