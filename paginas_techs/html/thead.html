Tag <thead> HTML

A tag thead é utilizada para criar o cabeçalho de uma tabela (<table>).

É recomendado criar as células no thead com a tag <th>.

<table>
    <thead>
        <th>Nome</th>
        <th>Idade</th>
        <th>Profissão</th>   
    </thead>

    <tbody>
        [...]
    </tbody>

    <tfoot>
        [...]
    </tfoot>
</table>
