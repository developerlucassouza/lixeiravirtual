Tag <pre> HTML

A tag pre irá fazer com que o texto no site fique igual ao texto no código, como quebras e espaçamentos.

<!-- Sem <pre> -->
<p>Olá,            Mundo!</p>
<!-- Com <pre> -->
<pre><p>Olá,            Mundo!</p></pre>

Olá, Mundo!
