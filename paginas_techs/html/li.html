Tag <li> HTML

A tag li é um item de lista, utilizado para adicionar itens as listas ordenadas (<ol>) e nas listas não ordenadas (<ul>)

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    <li>Item 4</li>
    <li>Item 5</li>
</ol>

<ul>
    <li>Item 1</li>
    <li>Item 2</li>   
    <li>Item 3</li>
    <li>Item 4</li>
    <li>Item 5</li>
</ul>
